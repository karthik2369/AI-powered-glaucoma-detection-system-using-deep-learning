Pediatric Glaucoma Detection

Our AI-driven platform is designed to assist early diagnosis of Pediatric Glaucoma by analyzing ocular images using Convolutional Neural Networks (CNN). Pediatric Glaucoma, a rare but serious condition in children, often leads to permanent vision loss if not detected early. With the help of a trained deep learning model, our system classifies eye images into normal or glaucoma-affected cases to support ophthalmologists in real-time screening. This project is implemented using Python, TensorFlow (CNN architecture), Flask for web interface, and MySQL for secure patient data management.
